# Todo Web Components Demo

A small vanilla JavaScript exercise for creating categories and tasks in the browser. Data is stored with `localStorage`, and custom elements are used for the task form and task list.

## What it demonstrates

- DOM updates and event handling
- Browser storage
- Custom elements
- Light and dark themes
- Keyboard-friendly form controls

## Run locally

Open `index.html` in a browser. No build step is required.

This is a learning exercise rather than a featured portfolio project.
